# 网络安全工具平台开发进程

## 项目概览

**项目愿景**: 基于云端部署的现代化网络安全工具平台  
**核心功能**: 多线程端口扫描器 + 网络质量监控器 + TCP通信实验室  
**技术栈**: Python FastAPI + Vue.js 3 + PySide6 + Docker  
**开发方法**: 测试驱动开发 (TDD)，测试覆盖率目标 >90%  
**部署目标**: 腾讯云服务器 (124.221.188.46)
